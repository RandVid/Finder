# ERD image (course deliverable)

To keep the repository small, the PNG/SVG diagram is **not** committed.

1. Open [dbdiagram.io](https://dbdiagram.io).
2. Paste the contents of `db/schema.dbml`.
3. Export PNG or SVG for your report and (optionally) add it locally — add `*.png` / `*.svg` to `.gitignore` if you do not want binaries in Git.
